# Autonomous Fruit Harvesting Robot 🍎🤖

This project presents an **Autonomous Agricultural Robot** designed to identify and harvest ripe fruits efficiently.

The robot navigates a predefined path using a **5-channel IR sensor array**, detects fruit ripeness using a **color sensor**, and performs harvesting using a **3-axis robotic arm controlled by ESP32**.

## Features
- Autonomous line navigation using IR sensors
- Fruit ripeness detection using RGB color sensor
- Smart decision making to pick or skip fruit
- 3-servo robotic arm for pick-and-place
- Motor driver based differential drive system
- Compact chassis optimized for farm navigation

## Hardware Used
- ESP32 Microcontroller
- IR Sensor Array (5 Channel)
- Color Sensor
- Servo Motors
- BO Motors
- Motor Driver
- Li-Ion Battery
- Custom 3D Printed Gripper
